# module-storage

Модуль хранения данных для системы управления задачами студентов (Student Task System).

## 📋 Описание

Данный модуль отвечает за сохранение, чтение, обновление и удаление данных задач. 
Он реализует базовые операции CRUD и обеспечивает надёжное хранение информации в формате JSON. 
Модуль не реализует бизнес-логику или пользовательский интерфейс — его задача предоставлять данные по запросу других компонентов системы.

## ⚙️ Функционал

Модуль предоставляет следующий интерфейс:

| Функция | Назначение | Возвращает |
|---------|------------|------------|
| `saveData(data)` | Сохранение новой задачи | `boolean` — успешность операции |
| `getData(id)` | Получение задачи по ID | `object` — данные задачи или `null` |
| `deleteData(id)` | Удаление задачи по ID | `boolean` — успешность удаления |
| `getAllData()` | Получение всех задач | `array` — массив всех записей |

## 📦 Формат данных

Все данные передаются в формате JSON-объектов со следующей структурой:

```json
{
  "id": 1,
  "title": "Название задачи",
  "deadline": "2026-05-01",
  "priority": 5
}
